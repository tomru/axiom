Form & Inputs

<TextInput>

Show <Base> properties
children
node

Optional TextInputIcon, all other children are ignored

disabled
bool

Disables interactions and applies styling

invalid
bool

Applies styling to indicate the users input was invalid

label
string

Descriptive label that is placed with the input field

placeholder
string

Descriptive placeholder text that is displayed in the input field when there is no value

size
enumDefault: medium

Size of the input field

style
enum

Visual style variations of the input field

type
enumDefault: text

Type of the input field

valid
bool

Applies styling to indicate the users input was valid

value
any

Value of the input field

onBlur
func

Handler for when the input field is blurred

onFocus
func

Handler for when the input field is focused

Component
unionDefault: div

The component that should be rendered with the Base component functionality. If a string is given then it must be a valid React registered element tag.

className
string

Class name to be appended to the element

hiddenUntil
enum

Control over when the element should be hidden until. Opposite of `visibleUntil`.

space
enum

Vertical margins given to the element

sticky
string

Distance from the top of the window where the element shall be stuck to. Utility for "position: sticky;".

textBreak
enum

Text wrap styling

textCase
enum

Text casing styling

textCenter
enum

Text central alignment either all of the time, with a value of `true` otherwise at one of the breakpoints specified.

textColor
enum

Text color styling

textEllipsis
bool

Text ellipsis styling

textEmphasize
bool

Text emphasis styling

textLeft
enum

Text left alignment either all of the time, with a value of `true` otherwise at one of the breakpoints specified.

textRight
enum

Text right alignment either all of the time, with a value of `true` otherwise at one of the breakpoints specified.

textSize
enum

Text size styling

textStrike
bool

Text strike styling

textStrong
bool

Text bold weight styling

textUnderline
bool

Text underlined styling

theme
enum

Theme styling modifier

visibleUntil
enum

Control over when the element should be visible until. Opposite of `hiddenUntil`.
