Base

Base component

The Base component provides a way to add global abilities to every component. For example text alignment. The idea is to remove the need to add generic CSS utility classes.

It is mainly used internally but can also be used like any other component.

I'm a center aligned subtly strong <header>
<Base Component="header" textCenter={true} textStrong={true} textSubtle={true}> ... </Base>
